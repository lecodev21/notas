# Archivos de ejemplo para importar

Selecciona todos los `.md` de esta carpeta (menos este README) y arrástralos
al botón de importar de la app.

## Qué notebook necesitas crear primero

Para que el importador asigne los notebooks correctamente, crea estos dos
notebooks en la app **antes** de importar:

| Notebook | Notas que irán ahí |
|----------|--------------------|
| `Trabajo` | sprint-planning · decision-arquitectura |
| `Proyectos` | ideas-app-movil |
| `Personal` | receta-pasta-al-limon · libros-pendientes |

Si el notebook no existe, la nota se importa igual pero sin notebook asignado.

## Tags que se crearán automáticamente

`trabajo` · `sprint` · `planificación` · `arquitectura` · `backend`
`ideas` · `producto` · `móvil` · `recetas` · `pasta` · `rápido` · `lectura` · `personal`
